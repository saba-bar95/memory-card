## Project: Memory Card

## This project is done as part of [Odin Course](https://www.theodinproject.com/)

Clicking on an image in a memory game gives you a score; however, clicking the same image twice resets the score.

## Build with

### Technologies

- React JS
- SCSS

### Tools

- Vite
- Visual Studio Code
- Git and Github
- NPM
- Eslint
- Prettier
